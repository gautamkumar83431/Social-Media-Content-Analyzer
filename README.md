Social Media Content Analyzer - Fullstack (React + Express)

Folders:
- server: Node/Express backend (uploads, PDF parse, OCR, history.json)
- client: React frontend (upload, analyze, history)

Run server:
cd server
npm install
npm start

Run client:
cd client
npm install
npm start

Server default: http://localhost:4000
Client default: http://localhost:3000
